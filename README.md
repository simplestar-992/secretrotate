# SECRETROTATE

**Encrypted Secret Rotation**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/secretrotate.git
cd secretrotate
go build -o secretrotate .
```

## Usage

```bash
./secretrotate --help
```
